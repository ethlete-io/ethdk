# Sort

Add sorting to an interactive element (eg. a table header).
Imported by adding the `SortImports` array to the `imports` array in your standalone component.

### Basic Usage

```html
<div etSort>
  <p et-sort-header>Sort header</p>
</div>
```

## In a table

```html
<et-table [dataSource]="source" (etSortChange)="sortSomething($event)" etSort>
  <ng-container etColumnDef="name" sticky>
    <et-header-cell *etHeaderCellDef et-sort-header> Sortable stuff </et-header-cell>
    <et-cell *etCellDef="let row"> {{ row.name }} </et-cell>
  </ng-container>

  <et-header-row *etHeaderRowDef="['stuff']" />
  <et-row *etRowDef="let row; columns: ['stuff']" />
</et-table>
```
