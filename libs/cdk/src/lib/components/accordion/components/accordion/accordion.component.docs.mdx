# Accordion

The accordion is a collapsible component that can be used to display content on demand.

### Basic usage

```html
<et-accordion label="Some accordion">
  <p>My content</p>
</et-accordion>
```

### Customizing the accordion label

A custom label template can be provided to the accordion component by adding a `ng-template` with the `et-accordion-label-wrapper` directive applied.
The `et-accordion-label` directive is optional. It only provides fitting styles to the label.

**Do not** place any interactive elements in the label template.

```html
<et-accordion>
  <ng-template et-accordion-label-wrapper>
    <span et-accordion-label>Custom label</span>
  </ng-template>

  <p>My content</p>
</et-accordion>
```

### Hint text

A hint can be displayed next to the accordion chevron icon.
It will be displayed on screen sizes from md and up.
The `et-accordion-hint` directive is optional. It only provides fitting styles to the hint.

**Do not** place any interactive elements in the hint template.

```html
<et-accordion label="Some accordion">
  <ng-template et-accordion-hint-wrapper>
    <span et-accordion-hint> Hint text </span>
  </ng-template>

  <p>My content</p>
</et-accordion>
```
