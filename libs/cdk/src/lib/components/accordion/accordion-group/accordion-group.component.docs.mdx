# Accordion group

The accordion group is a container for a group of accordion components.

### Basic usage

```html
<et-accordion-group>
  <et-accordion label="Some accordion">
    <p>My content</p>
  </et-accordion>
  <et-accordion label="Some accordion">
    <p>My content</p>
  </et-accordion>
  <et-accordion label="Some accordion">
    <p>My content</p>
  </et-accordion>
</et-accordion-group>
```
