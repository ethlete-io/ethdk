# Nav tabs

The nav tabs component is a container that holds a number of tabs.
It uses a `router-outlet` to display the content of the tab.

When the list of tab labels exceeds the width of the header, pagination controls appear to let the user scroll left and right across the labels.

### Basic usage

```html
<nav [tabOutlet]="tabOutlet" et-nav-tabs>
  <a et-nav-tab-link routerLink="./one">One</a>
  <a et-nav-tab-link routerLink="./two">Two</a>
  <a et-nav-tab-link routerLink="./three">Three</a>
  <a et-nav-tab-link routerLink="./four">Four</a>
  <a et-nav-tab-link routerLink="./five" disabled>Five</a>
</nav>

<et-nav-tabs-outlet #tabOutlet>
  <router-outlet></router-outlet>
</et-nav-tabs-outlet>
```
