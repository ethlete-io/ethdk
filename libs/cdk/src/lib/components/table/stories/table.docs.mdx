# Table

Display a table of data. This is a wrapper around the [Angular CDK Table](https://material.angular.io/cdk/table/overview).
Imported by adding the `TableImports` array to the `imports` array in your standalone component.

## Basic usage

```ts
mySource = [
  {
    name: 'Hydrogen',
    weight: 1.0079,
    symbol: 'H',
  },
];

colDefs = ['name', 'weight', 'symbol'];
```

```html
<et-table [dataSource]="mySource">
  <ng-container etColumnDef="name">
    <et-header-cell *etHeaderCellDef> Name </et-header-cell>
    <et-cell *etCellDef="let row"> {{ row.name }} </et-cell>
  </ng-container>

  <ng-container etColumnDef="weight">
    <et-header-cell *etHeaderCellDef> Weight </et-header-cell>
    <et-cell *etCellDef="let row"> {{ row.weight }} </et-cell>
  </ng-container>

  <ng-container etColumnDef="symbol">
    <et-header-cell *etHeaderCellDef> Symbol </et-header-cell>
    <et-cell *etCellDef="let row"> {{ row.symbol }} </et-cell>
  </ng-container>

  <et-header-row *etHeaderRowDef="colDefs"></et-header-row>
  <et-row *etRowDef="let row; columns: colDefs"></et-row>
</et-table>
```

## Sorting

You can either listen to the `etSortChange` event or bind a form control to `[sortControl]`

```html
<et-table [dataSource]="mySource" (etSortChange)="updateMySort($event)" etSort>
  <ng-container etColumnDef="name">
    <et-header-cell *etHeaderCellDef et-sort-header> Name </et-header-cell>
    <et-cell *etCellDef="let row"> {{ row.name }} </et-cell>
  </ng-container>

  <!-- shortend.... -->
</et-table>
```

## Misc

The table has a few other features that can be used.
This includes a loading state, no data row, and sticky columns.

```html
<et-table [dataSource]="mySource" [busy]="stuffIsLoading">
  <ng-container etColumnDef="name" sticky>
    <et-header-cell *etHeaderCellDef> Name </et-header-cell>
    <et-cell *etCellDef="let row"> {{ row.name }} </et-cell>
  </ng-container>

  <!-- shortend.... -->

  <ng-template etTableBusy>
    <div>
      <p>Loading...</p>
    </div>
  </ng-template>

  <ng-template etNoDataRow>
    <p>No data found</p>
  </ng-template>
</et-table>
```
