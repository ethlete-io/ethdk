# Number Input

The number input component is a wrapper around the native HTML number input element. It is used to allow users to enter a number.
Imported by adding the `InputImports` array to the `imports` array of your standalone component.

## Usage

```html
<et-input-field [formControl]="fg">
  <et-number-input />
  <et-label>Number input</et-label>
</et-input-field>
```
