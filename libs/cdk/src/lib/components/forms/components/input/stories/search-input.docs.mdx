# Search Input

The search input component is a wrapper around the native HTML search input element. It is used to allow users to enter a search text.
Imported by adding the `InputImports` array to the `imports` array of your standalone component.

## Usage

```html
<et-input-field [formControl]="fg">
  <et-search-input />
  <et-label>Search input</et-label>
</et-input-field>
```

## Adding a clear button

```html
<et-input-field [formControl]="fg">
  <et-search-input>
    <et-search-input-clear *etIfInputFilled etInputSuffix>clear</et-search-input-clear>
  </et-search-input>
  <et-label>Search input</et-label>
</et-input-field>
```
