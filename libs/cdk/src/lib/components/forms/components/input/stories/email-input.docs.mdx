# Email Input

The email input component is a wrapper around the native HTML email input element. It is used to allow users to enter an email.
Imported by adding the `InputImports` array to the `imports` array of your standalone component.

## Usage

```html
<et-input-field [formControl]="fg">
  <et-email-input />
  <et-label>Email input</et-label>
</et-input-field>
```
