# Tel Input

The tel input component is a wrapper around the native HTML tel input element. It is used to allow users to enter a telephone number.
Imported by adding the `InputImports` array to the `imports` array of your standalone component.

## Usage

```html
<et-input-field [formControl]="fg">
  <et-tel-input />
  <et-label>Tel input</et-label>
</et-input-field>
```
