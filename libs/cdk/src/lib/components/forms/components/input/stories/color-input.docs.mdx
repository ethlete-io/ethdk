# Color Input

The color input component is a wrapper around the native HTML color input element. It is used to allow users to open a color-picker and select a color.
Imported by adding the `InputImports` array to the `imports` array of your standalone component.

## Usage

```html
<et-input-field [formControl]="fg">
  <et-color-input />
  <et-label>Color input</et-label>
</et-input-field>
```
