# Password Input

The password input component is a wrapper around the native HTML password input element. It is used to allow users to enter a password.
Imported by adding the `InputImports` array to the `imports` array of your standalone component.

## Usage

```html
<et-input-field [formControl]="fg">
  <et-password-input />
  <et-label>Password input</et-label>
</et-input-field>
```

## Adding a show/hide button

The `et-password-input-toggle` component can be used to show or hide the password.
It can be either a prefix or suffix of the input field.

```html
<et-input-field [formControl]="fg">
  <et-password-input>
    <et-password-input-toggle etInputSuffix>show</et-password-input-toggle>
  </et-password-input>
  <et-label>Password input</et-label>
</et-input-field>
```
