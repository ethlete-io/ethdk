# Text Input

The text input component is a wrapper around the native HTML text input element. It is used to allow users to enter a text.
Imported by adding the `InputImports` array to the `imports` array of your standalone component.

## Usage

```html
<et-input-field [formControl]="fg">
  <et-text-input />
  <et-label>Text input</et-label>
</et-input-field>
```
