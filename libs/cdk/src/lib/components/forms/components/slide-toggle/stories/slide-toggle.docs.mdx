# Slide Toggle

The slide toggle component is a simple on/off toggle switch. It is used to quickly switch between two states.
Imported by adding the `SlideToggleImports` array to the `imports` array of your standalone component.

## Usage

```html
<et-slide-toggle-field [formControl]="fg">
  <et-slide-toggle />
  <et-label>Slide it</et-label>
</et-slide-toggle-field>
```
