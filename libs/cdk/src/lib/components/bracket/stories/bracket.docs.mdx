# Bracket

Display a bracket view of a tournament (single or double elimination).
Imported by adding the `BracketImports` array to the `imports` array in your standalone component.
Default configuration can be provided by adding `provideBracketConfig()` to a `providers` array. This allows for defining a match item & round header component.
Relevant information (such as match id or round details) will be injected into the respective components.

## Usage

Wrapping the bracket component in a scrollable component is recommended to avoid overflow issues.

```html
<et-scrollable stickyButtons>
  <et-bracket
    [itemHeight]="itemHeight"
    [itemWith]="itemWith"
    [columnGap]="columnGap"
    [rowGap]="rowGap"
    [roundsWithMatches]="roundsWithMatches"
    [roundHeaderHeight]="roundHeaderHeight"
    [upperLowerBracketGap]="upperLowerBracketGap"
  ></et-bracket>
</et-scrollable>
```
