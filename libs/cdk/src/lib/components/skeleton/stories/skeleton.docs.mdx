# Skeleton

Provide a placeholder while you wait for content to load, or to visualise content that doesn't exist yet.

## Code examples

Below you can find example code and usefull documentation for creating loading skeletons.

### Basic usage

```html
<et-skeleton loadingAllyText="Loading example thing...">
  <et-skeleton-item></et-skeleton-item>
  <et-skeleton-item></et-skeleton-item>
  <et-skeleton-item></et-skeleton-item>
</et-skeleton>
```

```scss
et-skeleton {
  background-color: #343434;
  width: 315px;
  padding: 16px;
  border-radius: 8px;
}

et-skeleton-item {
  background-color: #555555;

  &:nth-of-type(1) {
    width: 50px;
    height: 50px;
    border-radius: 50%;
    margin-bottom: 60px;
  }

  &:nth-of-type(2) {
    width: 60%;
    height: 24px;
    margin-bottom: 16px;
    border-radius: 4px;
  }

  &:nth-of-type(3) {
    width: 100%;
    height: 24px;
    border-radius: 4px;
  }
}
```

### Disable animations

The animations will always be disabled, if the user prefers [reduced motion](https://developer.mozilla.org/en-US/docs/Web/CSS/@media/prefers-reduced-motion).
It can also get explicitly disabled by setting the `animated` property to `false`.

```html
<et-skeleton loadingAllyText="Loading example thing..." animated="false">
  <et-skeleton-item></et-skeleton-item>
  <et-skeleton-item></et-skeleton-item>
  <et-skeleton-item></et-skeleton-item>
</et-skeleton>
```

## Styling

The skeleton component provides css classes & variables for styling.

#### `et-skeleton`

> The skeleton container.

<table>
  <tr>
    <td>Variable</td>
    <td>Description</td>
  </tr>
  <tr>
    <td>`--et-skeleton-gradient`</td>
    <td>The animated gradient that moves over the skeleton</td>
  </tr>
  <tr>
    <td>`--et-skeleton-animation-duration`</td>
    <td>The animation duration</td>
  </tr>
  <tr>
    <td>`--et-skeleton-animation-timing-function`</td>
    <td>The animation timing function</td>
  </tr>
</table>

#### `et-skeleton--animated`

> Applied to the skeleton container when the `animated` property is set to `true`.

#### `et-skeleton-item`

> The skeleton item.
