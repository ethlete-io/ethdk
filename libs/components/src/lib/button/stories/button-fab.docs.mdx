# FAB — Design Specification

Floating Action Button. Always circular; can expand to a pill with a label.

---

## Anatomy

```
[ icon ] [ label? ]           ← expanded=false: icon only
                               ← expanded=true:  icon + label (pill)
[ loader overlay ]            ← absolute, fades in during loading state
```

- **Icon slot** — `[etIcon]` attribute selector
- **Label** — hidden by default; revealed via width animation when `expanded=true`
- **Loader** — absolutely positioned spinner that replaces icon + label visually
- **Shape** — always fully circular (`border-radius: 9999px`), height matches width

Icon position is controlled by `iconAlignment` (`start` | `end`, default `start`).

## Focus ring

Rendered by `FocusRingDirective` — a separate visual layer outside the button's own box model. Shown only on keyboard focus (`:focus-visible`).

In Figma, implement the focus ring as an overlay frame placed around the button component, not as a property of the button itself.
