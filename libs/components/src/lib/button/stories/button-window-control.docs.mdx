# Window Control Button — Design Specification

Square icon-only button for toolbar and window chrome controls. No variants — single visual style with two kinds.

---

## Anatomy

```
[ icon ]                      ← always fills the button square
[ loader overlay ]            ← absolute, fades in during loading state
```

- **Icon slot** — direct child content; always `1em × 1em` (matches `--_et-window-control-button-icon-size`)
- **Loader** — absolutely positioned spinner that replaces the icon visually
- **Shape** — square, fixed width and height per size, border-radius 4 px

## Fallback colors (when no theme is provided)

When no color theme is applied, the button falls back to:

- Background: transparent
- Icon color: `rgba(255, 255, 255, 0.92)` (suitable for dark toolbar backgrounds)

## Focus ring

Rendered by `FocusRingDirective` — a separate visual layer outside the button's own box model. Shown only on keyboard focus (`:focus-visible`).

In Figma, implement the focus ring as an overlay frame placed around the button component, not as a property of the button itself.
