# Text Button — Design Specification

No background, no border. Text with an optional icon; underline on hover.

---

## Anatomy

```
[ icon ] [ label ] [ icon ]   ← iconAlignment: 'start' (default) | 'end'
         [ loader overlay ]   ← absolute, fades in during loading state
```

- **Icon slot** — `[etIcon]` attribute selector; always `1em × 1em` (relative to font-size)
- **Label** — inline flex container
- **Loader** — absolutely positioned spinner that replaces icon + label visually

## Focus ring

Rendered by `FocusRingDirective` — a separate visual layer outside the button's own box model. Shown only on keyboard focus (`:focus-visible`).

In Figma, implement the focus ring as an overlay frame placed around the button component, not as a property of the button itself.
