# Icon Button — Design Specification

Square icon-only button. Default variant is `transparent` (unlike the Surface Button which defaults to `filled`).

---

## Anatomy

```
[ icon ]                      ← always 1em × 1em
[ loader overlay ]            ← absolute, fades in during loading state
```

- **Icon slot** — `[etIcon]` attribute selector; always `1em × 1em` (relative to font-size)
- **Loader** — absolutely positioned spinner that replaces the icon visually

    <td>Higher opacity shift (see variant table)</td>
  </tr>
  <tr>
    <td>Disabled</td>
    <td>`cursor: not-allowed`; outline/tonal/transparent → 40% opacity; filled: no opacity change</td>
  </tr>
  <tr>
    <td>Loading</td>
    <td>Icon → opacity 0; spinner overlay fades in (absolute center)</td>

  <tr>

    <td>Switches visual variant (see Pressed State section)</td>
  </tr>

## Focus ring

Rendered by `FocusRingDirective` — a separate visual layer outside the button's own box model. Shown only on keyboard focus (`:focus-visible`).

In Figma, implement the focus ring as an overlay frame placed around the button component, not as a property of the button itself.
