# Textarea Input

The textarea input component is a wrapper around the native HTML textarea input element. It is used to allow users to enter a text.
Imported by adding the `InputImports` array to the `imports` array of your standalone component.

## Usage

```html
<et-input-field [formControl]="fg">
  <et-textarea-input />
  <et-label>Textarea input</et-label>
</et-input-field>
```
