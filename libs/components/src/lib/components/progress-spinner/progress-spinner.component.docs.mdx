# Progress spinner

A progress spinner is a circular indicator that shows that an operation is in progress.
It can either be determinate or indeterminate.

## Determinate progress spinner

```html
<et-progress-spinner value="50" />
```

## Indeterminate spinner

```html
<et-spinner />
```

## Indeterminate spinner (multi color)

```html
<et-spinner multiColor />
```
